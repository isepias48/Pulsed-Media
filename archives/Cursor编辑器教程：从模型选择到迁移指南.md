## 2. 从VS Code迁移

Cursor是基于VS Code的分支，因此拥有相似的界面和功能，同时专注于AI辅助编码。如果您已经是VS Code用户，迁移到Cursor将非常轻松。

以下是如何打开Cursor特定设置的方法：

- 单击右上角的齿轮按钮。
- 使用快捷键`Ctrl/⌘ + Shift + J`。
- 通过`Ctrl/⌘ + Shift + P`并键入“Cursor Settings”。

如果需要打开VS Code的特定设置，同样可以使用`Ctrl/⌘ + Shift + P`，然后键入“VS Code Settings”。
